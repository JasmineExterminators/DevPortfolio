Touchless Solitaire Through Brain Waves and Eye-Tracking

This is a game of solitaire that allows the player to play with thought and eye movement alone.
Tech Stack: Python, OpenCV, the Neurosity Crown, and CMU Graphics (a MVC graphics package that uses persistent objects).

Lawn-E: A Robotic Weeder

An automated, herbicide-free solution to uproot weeds from a lawn. Leverages computer vision to identify weeds and properly adjust the robot's position.
Top 15 at Hack-A-Startup Competition (Graduate Entrepreneurship Program)
Tech Stack: Solidworks CAD, Pytorch, Microcontrollers

Dance Dance Revolution on an FPGA with Verilog!

Built a DDR system using sensor pads, circuits, and an FPGA!
We learnt how to code in Verilog for the first time within one week during the Build18, 2025 Hackathon! From 0 exposure to completing a working project as freshmen!
We were also nominated for an award!

Tech Stack: Verilog, FPGAs, Circuitry

MEDucation

This app bridges the gap between medical professionals and patients. Patients are able to scan a pill bottle label, then our app outputs instructions and ingredients legibly and defines all medical jargon detected. Each year in the U.S, about 1.5 million PREVENTABLE medication errors occur.
1st place in senior division Technovation pitch competition
Tech Stack: API implementation, database manipulation, app UI/UX, camera capture, data extraction

Bubble App

Tech Stack: XCode, Swift

Tractive System Status Indicator
